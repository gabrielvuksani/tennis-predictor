# Tennis Predictor Design System

## Visual Identity

Bloomberg-style analytical trading dashboard. Orange accent, JetBrains Mono for data, Inter for UI.

## Token Definitions

All tokens defined as CSS custom properties in `src/tennis_predictor/web/generate.py` → `_write_css()`.

### Dark Theme (default)

--bg: #050505;       --surface: #080808;    --card: #0a0a0a;

--t: #e5e5e5;        --t2: #888;            --t3: #555;          --t4: #333;

### Typography
- **UI**: Inter (Google Fonts), weights 400-900
- **Data/numbers**: JetBrains Mono (Google Fonts), weights 400-700
- Scale: 8px (labels) → 9px (meta) → 11px (body) → 14px (names) → 22px (metrics) → 36px (probability)

### Spacing & Radius
- Sidebar: 280px (desktop), 220px (tablet)
- Card padding: 10-16px, Section gaps: 6-12px
- Radius: 4px (badges) → 8px (items) → 10px (panels) → 14px (containers)

## Layout Architecture

- **Desktop (>960px)**: `grid-template-columns: 280px 1fr` — sidebar + detail panel
- **Tablet (769-960px)**: Sidebar narrows to 220px
- **Small Tablet (481-768px)**: Sidebar becomes horizontal scroll strip
- **Mobile (<480px)**: Two-state slide transitions, bottom tab nav

## Responsive Breakpoints

| Breakpoint | Layout Change |
|------------|---------------|
| 960px | Sidebar 220px, metrics 3-col |
| 768px | Horizontal chip strip, detail grid 1-col |
| 480px | Mobile containers, bottom nav, slide transitions |
